Create branch on github account:
1. HelloWiseCode  //master branch
2. FeaturesOdataBranch //Child branch of the HelloWiseCode master branch

Open git terminal on repo directory;

use basic configuration.

1. git --version
2. git config --global user.name "avinash-kurhe"
3. git config --global user.email "kurheavinash@gmail.com"
4. mkdir myproject
5. cd myproject/
6. git init
7. git clone https://github.com/avinash-kurhe/HelloWiseCode.git
8. git branch //current branch name showing
9. git branch -a   //list of branches showing 
10. git checkout FeaturesOdataBranch  //change the branch from master to 'FeaturesOdataBranch'
11. git pull //fetch the current chages from the branch
12. git status
13. git add --all // added all stage on stage area
14. git status
15. git commit -m "Using the git terminal bash message"  // commited the changes on branch
16. git push origin FeaturesOdataBranch  // It's more important steps  to push on github branch , we need password of github account.
